# TuneBrush 🪥

**One tap. Two minutes. Brushing with a beat.**

A PWA that plays a 2-minute song while you brush your teeth. No more guessing when to stop — the music is your timer.

**[Try it live →](https://orsanable.github.io/tunebrush/)**

## How it works

1. Tap the play button
2. A 2-minute track starts playing
3. A circular progress ring fills as time passes
4. Song ends = brushing done

## Features

- PWA — install to home screen, works like a native app
- Offline support via service worker  
- Circular progress ring timer
- Synced lyrics display
- Dark minimal UI

## Tech

Single HTML file. No dependencies. No build step. Vanilla JS + Web Audio API.

## Install

Open the live link on your phone and tap "Add to Home Screen" for the full experience.
